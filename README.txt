
# NZM MODS IPTV Web App

This is a responsive, mobile- and PC-friendly IPTV player built for NZM MODS.

## Features
- Works in mobile and desktop browsers
- Channel search and filtering by category
- Favorite channels saved with localStorage
- Recently watched channels (5 most recent)
- Dark mode toggle
- HLS streaming via Hls.js

## How to Use
1. Open `index.html` in your browser OR upload everything to GitHub Pages.
2. Your channels are loaded from `channels.json` (generated from your M3U8 file).
3. Click a channel card to start playing.
4. Mark favorites with the star icon (â˜…).

## Customization
- To update the channel list, edit `channels.json`.
- Logos, names, and categories can all be customized.

## Hosting on GitHub Pages
1. Push this project to a GitHub repository.
2. Go to Settings > Pages.
3. Set the branch to `main` and folder to `/root`.
4. Your site will be live at `https://yourusername.github.io/yourrepo/`.
