
# NZM MODS IPTV Web App

This is a Netflix-style IPTV player that loads channels from `channels_from_latest_m3u.json`.

## Features:
- Sticky video player on top
- Search and filter by categories
- Dark mode and mobile-friendly
- Works on Android & PC browsers

## How to Run:
1. Use a local server (like Python or VSCode Live Server):
   python3 -m http.server

2. Open http://localhost:8000 in your browser

## Files:
- index.html
- style.css
- script.js
- channels_from_latest_m3u.json
