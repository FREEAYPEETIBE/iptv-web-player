# NZM MODS IPTV

A web-based IPTV player with a Netflix-style interface. Works in any modern browser (desktop, mobile, or TV).

## Features

- Clean dark mode layout
- Autoplay with HLS.js support
- Search and filter channels
- Works online or offline

## How to Use

### Option 1: Run Locally
1. Extract the ZIP file.
2. Open `index.html` in your browser.
3. Click on any channel to start streaming.

### Option 2: Host on GitHub Pages
1. Upload all files to a GitHub repository.
2. Go to `Settings > Pages` in the repo.
3. Set the branch to `main` and folder to `/ (root)`.
4. Access the site using the provided GitHub Pages link.

## Customize Channels
Edit the `channels.js` file to:
- Add or remove channels
- Update logos, URLs, or groups
